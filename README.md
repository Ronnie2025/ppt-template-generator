# PPT Template Generator

Free browser-only generator for PPT layout plans, AI presentation prompts, page-by-page design requirements, and delivery checks.

The tool is useful when a PowerPoint or AI presentation workflow needs a clear structure before using Gamma, Canva, PowerPoint Copilot, or another AI-PPT tool.

## What It Generates

- PPT layout table
- AI-PPT generation prompt
- Page-by-page design requirements
- Delivery checklist for facts, editability, AI tone, and client handoff
- Optional purchase note for the related Chinese AI-PPT delivery checklist pack

## Boundaries

- It does not upload user materials.
- It does not generate a downloadable PowerPoint file.
- It does not promise client approval, sales, orders, or business results.
- It is a prompt and checklist tool, not a done-for-you PPT service.
